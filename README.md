# Intelligent Traffic Flow Monitoring System

This project is an IoT-based traffic monitoring system that detects vehicle movement and traffic density using sensors and ESP32.

## Components
- ESP32
- Ultrasonic Sensor (HC-SR04)
- IR Sensor
- LEDs
- Breadboard

## Features
- Detects vehicle presence
- Measures traffic density
- Displays traffic condition using LEDs

## Technologies Used
- ESP32
- Arduino IDE
- IoT Sensors
